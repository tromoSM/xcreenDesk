# XcreenDesk
- share realtime screen and remote desktop for all users on your local network with no data fees.
## Installation 
- download the [latest windows installer](https://github.com/tromoSM/xcreenDesk/releases/download/v1.0/XcreenDesk-windows-installer.exe) from the [release page](https://github.com/tromoSM/xcreenDesk/releases/latest/)
- or run this command on your cmd
```bash
winget install xcreenDesk
```
- install
## Features
- No data fees
- Messaging in between the host and the viewers
- Toggle messaging and remote control
- See who's watching
- No limits of viewer count nor screenshare duration
- Banning and unbanning users
- Screencapture save for viewer
## Debugging
- Download the [latest windows installer with console](https://github.com/tromoSM/xcreenDesk/releases/download/v1.0/XcreenDesk-windows-installer-with-console.exe) and install
